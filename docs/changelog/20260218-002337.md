## Key Design Decisions
- **Restricted content strategy**: forward_messages → send_file(media ref) → download+re-upload
- **Album handling**: 1.5s debounce window to collect grouped messages
- **Session management**: async_sessionmaker, one session per forward operation
- **Config**: YAML for forward pairs, .env for credentials, auto-sync to DB on startup

# 添加固定密码登录鉴权

## 需求
- 系统部署后所有 API 端点完全裸露，无任何访问控制
- 需要密码保护，防止未授权访问

## 实现方案
- JWT Token + 固定密码鉴权
- 密码通过 SHA256 哈希比对
- Token 有效期 7 天

## 后端变更
- [x] 新增 `core/security.py` — JWT 生成/验证、密码校验、FastAPI 鉴权依赖
- [x] 新增 `api/login.py` — POST /api/login 登录接口
- [x] 更新 `core/config.py` — 添加 auth_secret_key 配置
- [x] 更新 `main.py` — 所有路由添加 JWT 鉴权依赖

## 前端变更

## 鉴权流程
1. 用户访问 → 无 Token → 重定向到 /login
2. 输入密码 → POST /api/login → 返回 JWT Token
3. Token 存入 localStorage → 后续请求自动携带 Authorization: Bearer
4. Token 过期或无效 → 401 → 清除 Token → 重定向到 /login

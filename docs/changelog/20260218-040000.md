# 视频元数据保留 + 日志系统

## 需求
- S3 下载重传丢失视频封面、时长、宽高等元数据
- 日志不够详细，无法排查问题
- 需要在网页端查看最近日志

## 变更列表

### 视频元数据保留
- [x] 新增 `_extract_media_attrs()` 提取视频/音频属性（时长、宽高、封面、文件名）
- [x] 新增 `_download_thumb()` 下载缩略图
- [x] S3 重传时传入 thumb、duration、width、height、supports_streaming 等参数

### 日志完善
- [x] forwarder 每条消息标注 Rule ID + 策略级别（S1/S2/S3/S4）
- [x] syncer 增加跳过计数 + 完成统计

### 网页端日志
- [x] 新增 `core/log_buffer.py` — 内存环形缓冲区（500 条）
- [x] 新增 `api/logs.py` — GET /api/logs 接口
- [x] 新增 `pages/Logs.tsx` — 日志页面（自动刷新 + 级别过滤）
- [x] 侧边栏增加 Logs 入口

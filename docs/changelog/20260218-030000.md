# 添加 Docker 部署支持

## 需求
- 项目需要 Docker 容器化部署能力

## 实现
- 四容器架构：frontend (nginx) → backend (uvicorn) → postgres + redis

## 变更列表
- [x] 新增 `backend/Dockerfile` — Python 3.11 镜像
- [x] 新增 `frontend/Dockerfile` — 多阶段构建 Node → nginx
- [x] 新增 `frontend/nginx.conf` — 静态文件 + 反向代理
- [x] 新增 `.dockerignore` — 排除无关文件
- [x] 新增 `.env.example` — 环境变量模板
- [x] 更新 `docker-compose.yml` — 添加 backend + frontend 服务
- [x] 更新 `pyproject.toml` — 添加 PyJWT 依赖

## 部署命令
```bash
cp .env.example .env  # 编辑填入 API_ID 和 API_HASH
docker compose up -d
```
